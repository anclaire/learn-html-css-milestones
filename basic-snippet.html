The snippet:

Want to search the web? Try Google!

    The Tags:
  • "p" and "/p"
  • "a" and "/a"
  • I'm not sure if class and id and target count as tags. They weren't according to this website.

The p tag:

  • w3schools
  • mdn web docs

The a tag:

  • w3schools
  • html.com

The a tag is a child of the p tag because it is nested inside of it. So the link appears within the paragraph because it is underneath is and indented.

The p tag needs an opening and closing cue (the /p) to signify where the paragraph begins and ends. Without the /p, the computer won't know where the paragraph ends.

    Attribute names
  • href
  • id
  • class
  • target
    Attribute values
  • href value = "https://google.com"
  • id value = "the-best-link"
  • class value = "banana"
  • target value = "blank"
    Attribute/value pair purpose
  • href value = "https://google.com": provides a link to google.com
  • id value = "the-best-link": the id attribute is unique and can only be used one per html document. It can be used to identify a specific element in a stylesheet or java script.
  • class value = "banana": a class is also an identifier that can be used in a stylesheet, but it can be used multiple times throughout a document rather than only once.
  • target value = "blank": the target element specifies that this link should open in a new window
